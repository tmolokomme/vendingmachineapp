# Spring boot + React and Redux example 

### Install

```sh
$ npm install
$ mvn clean install
```

### Run application during dev

#### run server first
```sh
$ java -jar target/vendingmachineapp-0.0.1.jar
```

or by running Application class from your IDE.

#### run hot reload server
```sh
$ npm start
```

#### run front-end tests
```sh
$ npm test
```
